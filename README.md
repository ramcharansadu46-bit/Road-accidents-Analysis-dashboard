# 🚨 Road Accident Analysis Dashboard (2010–2015)

## 📊 Project Overview
This project analyzes road accident data from 2010 to 2015 using Power BI and Python. The objective is to identify patterns, trends, and high-risk regions to support data-driven decision making.

## 🛠 Tools Used
- Power BI
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- DAX

## 📁 Dataset Features
- State
- District
- Year
- Number of Accidents
- Fatalities
- Injured

## 📌 Key Insights
- Identified high accident states and regions
- Observed increasing trend in accidents over years
- Region-wise distribution analysis
- District-level accident patterns

## 📈 Dashboard Features
- KPI Cards (Total accidents, fatalities, etc.)
- Year-wise trend analysis
- State-wise comparison
- Region-wise pie chart
- Interactive filters (Year, State, Region)
